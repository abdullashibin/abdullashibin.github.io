Key Features

  • Functional Blog Section – Integrated WordPress's built-in blogging functionality, making it easy to add and manage blog posts.
  • Fully Responsive & SEO-Friendly – Ensured the theme is mobile-friendly and optimized for search engines.

Technical Stack

  • WordPress Theme Development
  • PHP & WordPress Templating
  • Custom Post Types (CPT)
  • Advanced Custom Fields (ACF)
  • HTML, CSS, JavaScript
